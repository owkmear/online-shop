# Интернет-магазин

Написан на ASP.NET MVC и Ext JS (4.2.1). База данных сгенерирована с использованием Entity Framework Code First, использованы паттерны Unit Of Work и Repository

## Функционал

Реализовано две роли: пользователь и администратор. Роль администратора:
* добавление, редактирование и удаление товаров
* добавление, редактирование и удаление пользователей
* подтверждение и закрытие заказа

Роль пользователя:
* просмотр товаров
* наполнение корзины
* оформление, просмотр статуса и удаление заказа

Реализована стандартная ASP.NET авторизация. Для доступа использовать следующие аккаунты: admin, user1, user2, user3 с паролями 123456.
